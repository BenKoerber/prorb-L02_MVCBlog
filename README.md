1. Setup and start

```
# Clone repo
git clone git@github.com:BenKoerber/prorb-L02_MVCBlog.git
cd prorb-L02_MVCBlog

# Set everything up (install gems, setup db)
bin/setup

# Optional: populate with sample data
rails db:reset

# Start server
bin/dev
```
